## 使用说明 

基于Webpack的vue的脚手架

### 文档结构
 ![image](http://www.umetrip.com/gitImg/tree.png)
 
* build    ...........................webpack配置值
* config    ...........................全局变量配置
* src       ...........................源文件
* src/asset ...........................资源文件
* src/components  .....................组件
* src/components/common  ..............通用组件
* src/components/page  .....................页面
* src/data      ...........................数据文件
* src/router    ...........................路由配置

### 命令
1.  npm install  安装依赖
2.  npm run dev  运行开发环境
3.  npm run build  运行生产环境
